# Snake Game

A classic Snake game built with HTML5 Canvas and vanilla JavaScript.

## Play

Open `snake-game.html` in any modern web browser.

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move the snake |
| SPACE | Pause/Resume game |

## Features

- Smooth snake movement with glowing visual effects
- Arrow keys or WASD for controls
- Random food generation
- Snake grows longer when eating food
- Score increases by 10 points per food
- Game speeds up slightly as you score more
- Pause functionality
- Game over on wall or self collision

## Development

This is a pure HTML/JS project with no build step required.
